## 软件简介

**随舞音频视频生成器**是一款专为"随机舞蹈"活动设计的自动化工具。它根据给定的歌单，自动下载/读取音视频素材，按指定时间段剪辑，并在每首歌之间插入自定义间隔（空白/倒计时），最终合并为一段完整的音频（MP3）或视频（MP4）文件。

### 核心功能

- 支持 **B站在线下载**或**本地文件**两种素材来源
- 自动按歌单中指定的起止时间精确剪辑
- 曲间可插入**空白静音**、**音频倒计时**或**视频倒计时**
- 同时支持输出 MP3（纯音频）、MP4（带画面视频）或 ~~两个同时输出~~
- 音量均衡处理，避免歌曲响度差异过大

## 素材来源设置

### B站下载（默认）

选择「B站下载」后，软件会根据歌单中的 **BV号** 自动从B站下载对应视频/音频。下载的文件会缓存在临时目录中，**再次生成同一首歌时自动跳过下载**。

### 本地文件

选择「本地文件」后，点击「浏览」选择一个包含本地音频/视频文件的文件夹。软件会根据文件名的关键词自动匹配歌单中的歌曲。

- 支持的格式：`.mp3`、`.mp4`、`.m4a`、`.flac`、`.wav` 等
- 文件名包含歌单中的曲名即可匹配成功

## 输出设置

| 选项 | 说明 |
|------|------|
| **MP3** | 生成纯音频文件，适合活动播放 |
| **MP4** | 生成带画面的视频文件，适合投影/屏幕播放 |
| **两者** | ~~同时生成 MP3 和 MP4，各自独立; 选择同时生成时 曲间设置 一定要选 视频放倒计时~~|
| **音量均衡** | 对所有音频片段做响度归一化，避免曲目之间音量忽大忽小。**推荐开启** |

---

## 曲间设置

控制每两首歌之间的间隔处理方式：

| 选项 | 说明 |
|------|------|
| **间隔秒数** | 两首歌之间的间隔时长，范围 0~30 秒，默认 2 秒 |
| **空白** | 间隔期间为纯静音/黑屏（默认） |
| **音频倒计时** | 间隔期间播放倒计时音效（需 `assets/5秒音频频计时.mp3`） |
| **视频倒计时** | 间隔期间播放倒计时视频（需 `assets/5秒倒视频计时.mp4`） |

选择「音频倒计时」或「视频倒计时」时，倒计时素材时长不足的部分会自动用空白补齐。

## 歌单管理

### 歌单文件格式（CSV）

歌单为 CSV 格式（逗号分隔），编码为 **UTF-8**，包含以下列：

| 列 | 说明 | 示例 |
|----|------|------|
| 曲名 | 歌曲名称 | `Drama` |
| BV号 | B站视频BV号 | `BV1xx411c7mD` |
| 开始时间 | 剪辑起始时间（秒） | `00:00:10` |
| 结束时间 | 剪辑截止时间（秒），留空表示播放到结尾 | `00:01:02` |


**示例歌单文件 `songlist.csv`：**

```csv
曲名,BV号,开始时间,结束时间
Drama,BV1xx411c7mD,35.5,65.0
Supernova,BV1abc123456,10.0,40.0
How Sweet,BV1def789012,0,30.0
Magnetic,BV1ghi345678,60.0,
```

> 结束时间留空则播放整首歌（从起始时间到结尾）。

### 加载歌单

1. 点击「浏览」选择歌单 CSV 文件
2. 软件自动读取并显示在下方表格中
3. 表格显示：**曲名**、**剪辑时间**、**BV号**
4. 点击「重新加载」可刷新歌单内容

## 生成与停止

### 开始生成

1. 确认设置无误后，点击底部的「🚀 开始生成」按钮
2. 软件按顺序处理每首歌：
   - B站模式：下载 → 剪辑；本地模式：查找文件 → 剪辑
   - 每首歌之间插入曲间间隔
   - 全部处理完毕后合并输出
3. 进度条和日志区实时显示处理状态
4. 处理完成后弹窗提示，自动打开输出目录

### 停止生成

点击「⏹ 停止」可随时取消当前任务。已处理的文件会保留在临时目录中，下次重新生成会更快。

## 输出目录

所有生成的文件保存在软件目录下的 `output` 文件夹中：

```
output/
├── .temp/                    ← 临时文件（剪辑片段、下载缓存）
├── 随机舞蹈_20260612_1630.mp3  ← 最终生成的音频
└── 随机舞蹈_20260612_1630.mp4  ← 最终生成的视频
```

点击「📂 打开输出目录」可快速打开此文件夹。

## 常见问题

### Q: B站下载失败怎么办？

**A:** 
1. 确认 BV 号正确无误
2. 确认该视频在 B站上可公开访问
3. 检查网络连接是否正常
4. 日志区会显示具体失败原因

### Q: 提示"未找到 FFmpeg"？

**A:** 请确保 `assets` 文件夹与程序在同一目录，且包含 FFmpeg 可执行文件。

### Q: 生成速度很慢？

**A:** 
- 首次下载 B站视频需要时间，二次生成时缓存命中会快很多
- 视频生成（MP4）比音频（MP3）慢，如只需音频建议选 MP3
- 可以关闭不必要的输出格式（如只选 MP3 而非两者）


### Q: 歌单中某首歌"未找到本地文件"怎么办？

**A:** 
1. 确认本地文件夹中有对应歌曲文件
2. 文件名需包含歌单中的曲名关键词
3. B站模式下请确认 BV 号正确
